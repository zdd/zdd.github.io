undefined vs not defined

In JavaScript, undefined and not defined are two different concepts.

  • undefined: a variable has been declared but has not yet been assigned a value.
  • not defined: a variable has not been declared(not exists).

not defined

Whenever you try to access a variable that is not declared, JavaScript throws an error: Uncaught ReferenceError: xxx is not defined. This is because the variable b is not declared anywhere in the code. but you can still use typeof to check if a variable is defined or not.
