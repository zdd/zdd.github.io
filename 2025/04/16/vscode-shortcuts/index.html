VSCode是如今市面上最流行的IDE了，用好VSCode是每一个前端程序员必备的技能，而VSCode又有大量的快捷键可以使用，下面是一些常用的VSCode快捷键。

说明

在MacOS上请将Ctrl替换为Command，Alt替换为Option

Run command

使用Ctrl + Shift + P：打开命令面板，然后可以输入一下命令进行相关操作

将VSCode加入环境变量

install code command in PATH：将VSCode加入环境变量，方便在终端中使用code命令打开VSCode - 注意此法需要管理员权限，很多公司配的电脑没有管理元权限。那么只能通过环境变量的方式添加
